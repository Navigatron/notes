Do-While loop

Java:

do{
    // Code
}while(condition);

MIPS pseudo Code:

loop:
    # body
    # update
    if(condition) goto loop

or

    addi    $s0, $zero, 0
loop:
    # body
    addi    $s0, $s0, 1
    slti    $t0, $s0, 10
    bne     $t0, $zero, loop

Do while loops are real tasty. But if it can't be persuaded into a do-while, don't force it.

Switch Statements

switch(s0){
    case 0:
        // code
        break;
    case 1:
        // code
        break;
    default:
        // code
}

Just a bunch of if-else stuff - kinda. Switch has more requirements than if, but runs faster.
